# About
> Private School is a C++ console application for managing a private school.

## What you can do
- Create or load a school
- Edit the school name
- Add classrooms to the school
- Manage classrooms and add students
- View school summaries, classroom lists, student lists, and statistics
- Delete the saved school data and start over

## How it works
> The current persistence layer saves the school name to `data/school_data.txt`.
> Classrooms and students are managed in memory during the session.

## Navigation
- Use `ABOUT` to open this page
- Use `MANAGE SCHOOL` to configure the school
- Use `VIEW REPORT` to browse reports
- Use `EXIT` to close the program
