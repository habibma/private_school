# Private Teacher 🧑‍🏫

**Private Teacher** is a C++ program designed to help private teachers manage their classrooms.  
This program helps a teacher keep track of their students, store names and scores, and calculate class averages.

## ✨ Features
- Register teacher’s name and surname
- Add students with their names and scores
- Validate inputs (only letters for names, only numbers for scores)
- Calculate and display class average
- Designed to be extended for:
  - Private teacher management
  - Private school management
  - Parents as teachers

## 🚀 Getting Started

### Prerequisites
- A C++ compiler (e.g., `g++`, `clang++`)
- Git (to clone the repo)

### Build and Run
```bash
git clone https://github.com/habibma/private_teacher.git
cd private_teacher
make
./program
```
